## Http Methods描述

| HTTP Methods	| 操作方式 |
| ---- | ---- | ---- | ---- | 
| POST | 创建数据 Create	 |
| GET | 读取数据 Read |
| PUT | 修改数据 Update 整条修改修改除ID外的所有属性 |
| PATCH	| 修改数据 Update 部分修改 修改一条记录的部分属性 |
| DELETE | 删除数据 Delete |
